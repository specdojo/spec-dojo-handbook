# 業務データ辞書

業務データ辞書は、業務で扱うデータの意味・定義・構造を定義する仕様書群です。

## ドキュメントの一覧

| ドキュメント         | 略称 | 主な対象                     | 内容                                                                               | 関連テンプレート                                          |
| -------------------- | ---- | ---------------------------- | ---------------------------------------------------------------------------------- | --------------------------------------------------------- |
| 業務データ辞書       | BDD  | データ項目                   | 業務上の管理単位（エンティティ）とその属性の論理名・物理名・説明・制約などを定義。 | [ルール](bdd-rules.md) / [作成指示](bdd-instruction.md)   |
| 概念データストア一覧 | CDSL | データストア（概念レベル）   | 業務で扱うデータストア（帳簿・台帳・記録など）を一覧で定義。                       | [ルール](cdsl-rules.md) / [作成指示](cdsl-instruction.md) |
| 保管場所一覧         | SLL  | 物理・論理の保管場所         | 業務で管理する物理的・論理的な保管場所を一覧で定義                                 | [ルール](sll-rules.md) / [作成指示](sll-instruction.md)   |
| ステータス           | STS  | 状態（業務・システム）       | 業務上のエンティティが取り得る状態（ステータス）を一覧で定義                       | [ルール](stl-rules.md) / [作成指示](stl-instruction.md)   |
| 分類一覧             | CLS  | 分類（カテゴリ・種別・区分） | 業務上の分類（カテゴリ、種別、区分など）を一覧で整理                               | [ルール](cll-rules.md)                                    |

## 利用ガイド

1. 各 `*-rules.md` にドキュメントの作成ルールが記述されています。
2. 生成AIを活用してドキュメントを作成する場合は `*-instruction.md` をプロンプトのテンプレートとして活用ください。
