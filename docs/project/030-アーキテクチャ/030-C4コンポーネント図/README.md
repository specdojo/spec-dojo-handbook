# C4コンポーネント図

主要コンテナ内のコンポーネント（責務のまとまり）と、外部要素との関係を、C4コンポーネント図として定義します。

## 記法ガイド

1. 図を含むドキュメント（Markdown）の作成・更新ルールは [cpd-rules.md](cpd-rules.md) を参照してください。
2. 生成AIを活用して Mermaid 図を含むドキュメント(Markdown)を作成する場合は [cpd-instruction.md](cpd-instruction.md) をプロンプトのテンプレートとして利用してください。
3. Mermaid 図の作図ルールは [cpd-mermaid-rules.md](cpd-mermaid-rules.md) を参照してください。
4. 生成AIを活用して Mermaid のC4コンポーネント図を作成する場合は [cpd-mermaid-instruction.md](cpd-mermaid-instruction.md) をプロンプトのテンプレートとして利用してください。
