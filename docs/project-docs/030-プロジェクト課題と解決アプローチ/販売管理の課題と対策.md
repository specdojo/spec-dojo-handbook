# 駄菓子屋きぬや販売管理の課題と対策

## 課題一覧

1. 人気商品の欠品が多い
2. 仕入れの勘が、忙しさや体調に左右される
3. 常連の「つけ」の記録がノートで散在し、たまに記録漏れがある
4. おばあちゃん以外が店番をすると、状況がよく分からない

## 課題1: 人気商品の欠品

- 原因
  - 在庫数と売れ行きの傾向が把握できていない
  - 発注のタイミングが商品ごとにバラバラ
- 解決アプローチ
  - 商品ごとに「発注点」と「定番商品」フラグを設定
  - 毎日の販売数を記録し、在庫が発注点を下回ったら「発注候補一覧」を表示

## 課題3: 「つけ」の記録漏れ

- 原因
  - ノートの書き忘れ・読み間違い
  - 家族が店番をしたときにルールが徹底されない
- 解決アプローチ
  - 顧客ごとに「つけ残高」をシステムで管理
  - 販売登録時に「現金／つけ」を必ず選択する画面設計
  - 顧客名／ニックネームでの検索・選択をサポート

## 全体アプローチ

- できるだけ「おばあちゃんの今のやり方」に寄せた画面・言葉づかいを採用
- 複雑な分析機能より、「毎日見る在庫と今日の売上」がすぐ分かることを優先
- 将来的な機能拡張（家族のスマホからの在庫確認など）を見据えつつ、初期はシンプルに
