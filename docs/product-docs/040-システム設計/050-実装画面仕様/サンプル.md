# 実装画面仕様（販売登録画面）

## 画面ID

- SCREEN-SALES-001

## ルーティング

- Path: `/sales/new`
- Method: GET

## 構造

- ヘッダ: 「今日の売上」へのリンク、「在庫一覧」へのリンク
- 販売フォーム
  - 商品検索コンポーネント
  - 選択済み商品リストコンポーネント
  - 支払い方法セクション
  - 顧客選択コンポーネント（支払い方法=つけ時のみ表示）
  - 合計金額表示
  - 「会計確定」ボタン

## 使用API

- `GET /api/products?keyword=...`
- `POST /api/sales`
- `GET /api/customers?keyword=...`
