# あらすじ

主人公はCoderDojoに通う小学６年生の天才少年、算野創太。
最近、おばあちゃんの物忘れが激しくなってきて、営んでいる駄菓子屋でいろいろな問題が発生するようになってきた。
主人公はおばあちゃんのために、駄菓子屋の運営を支援するシステムを作ることを決意する。
物語は、駄菓子屋の問題解決と主人公の成長を描くハートフルなストーリーとなる。
